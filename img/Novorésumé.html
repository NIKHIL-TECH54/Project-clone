How does it work?

You need to select a section from the left-hand side menu. Under that section, you can then add whatever data you believe you will need later or that you will use most often.

Afterward, whether you’ve just created a new document or you’re editing an existing one, you can easily pull up the „My Content” Quick Access, and select the data you want to be copied into your document.

Why is there a limit of entries under each section?

We’ve limited the amount of data that can be entered so that we keep the performance of the web application as high as possible.

What is My Content?

The „My Content” is your library of reusable data. Here you can easily create new entries which can then be copied into any document that you are working on. Think of it as a template of data that you can reuse where you see fit.

What happens if I delete data from My Content?

The data will be removed from the „My Content”, however, your existing document will not be affected by this.

What happens if I reached the limit of entries?

Once you reach the limit for a section, you will not be able to add more entries under that section unless you delete some existing ones.

Can I import data from documents into the „My Content”?

At the moment you can only manually add entries to the „My Content”.
